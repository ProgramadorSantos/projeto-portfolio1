O que é HTML5?

HTML é a sigla para "Hypertext Markup Language", que significa Linguagem de Marcação de Hipertexto em português. É uma linguagem de marcação usada para criar e estruturar conteúdo na web.

O HTML é a base para a criação de páginas e aplicativos web. Ele permite que os desenvolvedores definam o conteúdo de uma página e a forma como ele é apresentado ao usuário. Isso inclui o texto, imagens, links, formulários e outros elementos que podem ser exibidos em um navegador web.

A linguagem HTML é baseada em marcação de tags (etiquetas), que são elementos especiais que definem a estrutura e o significado do conteúdo. As tags são escritas usando uma sintaxe específica e devem ser colocadas corretamente para que o navegador possa renderizar corretamente a página.

O HTML também pode ser combinado com outras tecnologias, como CSS (Cascading Style Sheets) para definir a aparência visual de uma página, e JavaScript para adicionar interatividade e comportamento dinâmico à página.

Em resumo, o HTML é a linguagem de marcação que permite que os desenvolvedores criem conteúdo na web de forma estruturada e organizada, para que possa ser exibido de forma adequada e acessível aos usuários de um navegador web.

As principais partes de um elemento HTML são

  • 1ª A tag de abertura
  • 2ª Tag de fechamento
  • 3ª Conteúdo
  • 4ª Elemento

O que é CSS3

CSS é a sigla para "Cascading Style Sheets", que significa Folhas de Estilo em Cascata em português. É uma linguagem de estilo usada para definir a aparência visual de páginas web escritas em HTML ou outra linguagem de marcação.

O CSS permite aos desenvolvedores web separar a apresentação visual do conteúdo estruturado de uma página web. Isso significa que o CSS pode ser usado para definir o layout, a cor, a fonte e outros aspectos visuais de uma página web, sem afetar o conteúdo ou a estrutura da página em si.

As folhas de estilo em cascata são aplicadas a um documento HTML de cima para baixo, permitindo que os estilos sejam sobrepostos ou anulados conforme necessário. Isso significa que é possível criar estilos genéricos para elementos comuns em uma página e, em seguida, anular esses estilos para elementos específicos, se necessário.

O CSS também permite a criação de estilos responsivos, que mudam com base no tamanho da tela ou dispositivo em que a página é exibida, e a criação de animações e efeitos visuais dinâmicos.

Em resumo, o CSS é uma linguagem de estilo que permite aos desenvolvedores web controlar a aparência visual de uma página web, separando a apresentação visual do conteúdo estruturado de uma página web.

As principais partes de um elemento CSS são

  • 1ª Seletor
  • 2ª Propriedade
  • 3ª Valor
  • 4ª Declaração
